# Analytics Admin Dashboard

A sleek, modern analytics admin dashboard built in **pure HTML5, CSS3, and Vanilla JavaScript** with zero frameworks or build dependencies.

## 🚀 Files

- **`index.html`**: Main single-page application structure.
- **`style.css`**: Modern dark-themed glassmorphism design system.
- **`script.js`**: Lightweight JavaScript application logic and data handling.

## ✨ Features

- **Demo Authentication**: Pre-filled login screen (`admin` / `admin123`) with persistent session status in `localStorage`.
- **Interactive Dashboard**: Stat cards, Revenue line chart & Category sales bar chart powered by Chart.js CDN.
- **Users Directory**: Fetches user list live from JSONPlaceholder API with real-time search filtering and user add/delete modals.
- **Settings & Theme**: Persistent Light / Dark mode toggle.

## 💻 How to Run

Simply open `index.html` in any modern web browser.
